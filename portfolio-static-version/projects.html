E-commerce pizza ordering site (study case)

Software university (SoftUni)

February - April 2024

Tech Stack:

Back-End: NodeJS, Express

Front-End: Angular

Database: MongoDB

Key Contributions:

  • Choose to add a product to your cart
  • Select a quantity and proceed to checkout by entering a location with Google Maps and then using PayPal for payment. When payment is complete, an order details page will appear;
  • For login and registration, there is a validation of the fields and for an invalid page it redirects to the"Not Found" page;
  • Evaluation is done with ngBootstrap and popups with Toastr. Search for a product by name and there are tags by which it is possible to sort the products.

Software university (SoftUni)

October - December 2023

Tech Stack:

Back-End: interfacing with the SoftUni Practice Server
